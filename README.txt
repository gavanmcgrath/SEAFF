Download to a location on your computer.
Open R
It can be downloaded from https://cran.r-project.org/
At the console set the working directory to the SEAFF folder via typing the following at the R prompt and hitting enter:
setwd("C:/myRdir/SEAFF")
replacing C:/myRdir by the appropriate address.
The type:
source('SEAFF.R')

If this is your first tiem using R the scripts will attempt to download and install all the required R packages needed.
If successful your browser should open.
